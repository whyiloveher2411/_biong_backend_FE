# Vietnamese voiceover — Writing for the Ear

Read when writing **Vietnamese** narration for marketing short video (Biong HyperFrames pipeline). For OmniVoice expressive tags and HASCAS structure, use the Biong skill references — this doc covers spoken-language craft only.

Canonical full spec: `biong-short-video-hyperframes/references/vi-voiceover-naturalization.md` (in Biong FE repo).

---

## 1. Break written structure (eye → ear)

- Each spoken line **≤12 words** (Vietnamese words).
- **Ban** formal connectors: "do đó", "vì vậy", "tuy nhiên", "nhằm mục đích", "tiến hành".
- **Replace** with spoken Vietnamese:

| Written | Spoken |
|---------|--------|
| Do đó / Vì vậy | Nên là… / Thành ra… |
| Tuy nhiên | Nhưng mà… / Thế nhưng… |
| Tiến hành xử lý | Làm… / Bắt tay vào làm… |
| Phương pháp tối ưu | Cách tốt nhất / Mẹo cực hay… |
| Chúng ta sở hữu | Mình có… |

- No bullet lists read in one breath — split into short lines.

---

## 2. Conversational anchors (humanize step)

Add sparingly (2–4 per 60–90s video):

- **Openers:** Biết sao không?… / Nghe nè… / Bật mí nhé… / Thật ra thì…
- **Closers:** …nè / …đúng không? / …luôn á! / …thử xem sao nhé.

---

## 3. Retention pacing (HASCAS)

| Beat | Timing | Voice |
|------|--------|-------|
| Hook | First ~3s | Shock question or bold claim; use `?!` — **no ALL CAPS** |
| Agitate | ~3–15s into agitate block | Short exclamations; brand-safe ("Ức chế quá đi!") |
| Solve | ~60% of script | Neutral, clear steps |
| CTA | ~10% end | Confident, short |

Word budget: ~2.5 words/second for 60–180s scripts.

---

## Tone

- Read aloud. If it sounds like a brochure or academic paper, rewrite.
- Vary sentence length — mostly short, occasional slightly longer flow.
- Questions and exclamation marks drive TikTok/Reels energy.

## Pacing reference

- **2.5 words per second** natural Vietnamese pace
- 60s ≈ 150 words · 90s ≈ 225 words · 180s ≈ 450 words
